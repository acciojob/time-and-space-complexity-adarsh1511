Time Complexity and Space Complexity

Time complexity is defined as the amount of time taken by an algorithm to run, as a function of the length of the input. It measures the time taken to execute each statement of code in an algorithm. The algorithm that performs the task in the smallest number of operations is considered the most efficient one in terms of the time complexity.

Space complexity refers to the total amount of memory space used by an algorithm/program, including the space of input values for execution. O(n) – linear complexity – takes space directly proportional to the input size.
